# UI Prompt Baseline

Use this baseline when generating new screens:

- Follow the active token system from [DESIGN_SYSTEM.md](/C:/Users/quock/OneDrive/picklematch-vn/DESIGN_SYSTEM.md).
- Colors and semantics must come from `profileTheme -> constants/theme -> useAppTheme`.
- Reuse shared components in `components/design/*` whenever possible.
- Keep Vietnamese copy clean UTF-8.
- Avoid one-off palettes and ad-hoc visual systems.

Required output for AI mockups:
1. Mobile-first screens (iOS + Android).
2. Token usage mapping.
3. Reusable component variants.
4. Loading/empty/error/success states.
